GitHub Pages + GitHub API
Token safety: create a GitHub fine-grained token for this repository only, with Contents: Read and write. Do not commit the token anywhere. For better safety, set an expiration date.
  1. GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens.
  2. Select repository: Shirk2002/pet_catalog.
  3. Repository permissions → Contents → Read and write.
  4. Paste the token below, load catalog, edit, then save.
