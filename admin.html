Token 安全提醒：请只为当前仓库创建 GitHub Fine-grained Token，并授予 Contents: Read and write 权限。不要把 Token 提交到仓库，建议设置过期时间。
  1. GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens。
  2. 选择仓库：Shirk2002/pet_catalog。
  3. Repository permissions → Contents → Read and write。
  4. 把 Token 粘贴到下面，加载目录，编辑商品，然后保存。
